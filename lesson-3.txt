1.
- git commit --amend: Dùng để chỉnh sửa commit cuối cùng (Git mở editor)
- git commit --amend -m”<message>: Sửa commit message ngay trong dòng lệnh, không cần mở editor.
- git restore --staged: Đưa file từ staging area (đã git add) về lại working directory
- git reset HEAD~1: Undo commit gần nhất (hủy commit nhưng không mất file). (đưa file lại vùng Working directory)
